# Beebe Dad's Golf Outing Microsite

Open index.html in a browser to view the site.

## Updating the golfer and prize data

Most event data lives in data.js.

- Update golfer names, indexes, and course handicaps in the players list.
- Update the draft date or team names in the event section.
- Update buy-in, golfer count, and payouts in the prizePool section.

## Draft, matchups, and scoreboard

The draft board, matchup selections, and scoreboard save in the browser you use. There is no backend or database, so those entries are not shared automatically across other computers or browsers.
